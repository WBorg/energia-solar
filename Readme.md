
# Geração de energia Solar




## Instalação

Projeto criado utilizando o ViteJs

 npm 6.x
```bash
npm create vite@latest my-vue-app --template vue
```
 npm 7+, extra double-dash is needed:
```bash
npm create vite@latest my-vue-app -- --template vue

```

### para rodar o projeto
```bash
npm run dev
```
